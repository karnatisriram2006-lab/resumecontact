# **App Name**: ResumeContact Pro

## Core Features:

- Landing Page: Display a modern landing page with key features, CTAs, and testimonials to attract users.
- Interactive Resume Form: Provide a multi-step form for users to input personal details, education, skills, experience, and projects.
- Real-time Resume Preview: Show a dynamic, live preview of the resume as the user fills out the form, allowing them to see the final result instantly.
- Firestore Data Storage: Store submitted resume data in a Firestore collection for easy retrieval and management.
- Contact Form: Enable users to send contact messages, including optional file uploads (PDF/Doc), stored in Firestore.
- Admin Dashboard with Login: Create a protected admin dashboard with Firebase Auth to manage resume requests and contact messages.
- AI Enhanced Resume Suggestions Tool: Employ a tool to parse user inputs and recommend improvements for wording in the resume. Can suggest appropriate keywords or action verbs to use in a given industry. Generates possible formatting changes.

## Style Guidelines:

- Primary color: Deep blue (#1A5276) for professionalism and trust.
- Background color: Light gray (#F0F2F5) for a clean, modern look.
- Accent color: Soft orange (#E59866) for CTAs and interactive elements.
- Body font: 'Inter' (sans-serif) for a clean, readable interface.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-forward look.
- Use a set of simple, consistent icons to represent different sections and actions, improving usability and visual appeal.
- Implement a responsive grid layout for optimal viewing on mobile and desktop devices.
- Add subtle hover effects and transitions to create an interactive and engaging user experience.
